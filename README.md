# 影余寄情 - 李德胜摄影集

> 此册既是个人岁月的印记，亦是家人厚爱之见证

## 简介

《影余寄情》是李德胜先生退休后近二十年摄影生涯的作品集。摄影集收录约80幅摄影作品，涵盖自然风光、人文纪实、花鸟生态等多种题材，按主题分为三篇。

## 章节结构

### 第一篇：步履赴山河 · 光影记远途
山河风光与远途旅行摄影

### 第二篇：笑泪皆入镜 · 百态见真情
人文纪实与人物百态

### 第三篇：四时皆有景 · 花鸟尽含情
自然生态与花鸟摄影

## 本地预览

直接在浏览器中打开 `index.html` 文件即可预览。

## 发布到 GitHub Pages

1. 创建 GitHub 仓库
2. 上传所有文件（包括 `index.html`、`styles.css`、`script.js` 和所有图片）
3. 在仓库设置中启用 GitHub Pages（Settings → Pages → Source: Deploy from a branch → main → root）
4. 等待几分钟后即可通过 `https://你的用户名.github.io/仓库名/` 访问

## 技术栈

- HTML5 + CSS3 + JavaScript
- 纯前端实现，无需服务器
- 响应式设计，支持移动端浏览
